## 1. JIRA의 활용

1. 현업 + SSAFY에서 계속해서 사용할 JIRA에 대한 기초적인 이해
2. JIRA는 issue 관리와 일정 관리 도구로 사용

![Managing Jira Cloud Projects | Think&amp;Do](https://mk0thinkanddooqusnvl.kinstacdn.com/wp-content/uploads/2020/06/jira_large.png)

### 1.1. JIRA란

JIRA란 아틀라시안이 개발한 **사유 이슈 추적** 제품입니다. 

- 버그 추적, 
- 이슈 추적, 
- 프로젝트 관리 

- i.g.
  - 작업내용 - 우선순위, 작업순서도 작성
  - PMS : 프로젝트매니즈 서비스



### 1.2. JIRA 중요성

#### 1.2.1. Project Management

- Project Policy
- Permission
- Roll & Rule
- Issue Tracking

#### 1.2.2. 범용적으로 사용 중인 PMS 활용 경험

- 여러 기업에서 사용 중
- 취업 전 제대로 사용 경험을 쌓기 어려운 PMS
- 중요도
  - Critical
  - Major
  - Minor



### 1.3. JIRA의 기능

#### 1.3.1. 핵심 기능

- Backlog
  - 모든 이슈 티켓이 관리되는 공간입니다.
  - MileStone을 확인합니다.
- Active sprint
  - Sprint 도 들어와서 들어와서 진행상황 확인 및 앞으로 얼마나 남았는지 예상가능
  - pre 진행 전
  - ing 진행 중
  - post 진행 완료
- Workflow
  - To do(이슈등록) -> In progress (작업중) -> Done (완료)
  - burndown chart
    - 언제 어떻게 이슈가 끝났는지 그래프로 확인
    - 작업량에 대해 코멘트 추가 등
    - 팀 내 그라운드 룰을 만들어서 지라관리를 함
- Story poinjt
  - 피보나치 수열 수
  - 난이도 기준 : Junior / Senior 할당받는 작업이 다름
